Kaggle Group Project: Car Prices

A group project with my peers. Regression analysis of used cars prices in Belarus found on Kaggle using R.

Data for this project is sourced from Kaggle (Available at: https://www.kaggle.com/lepchenkov/usedcarscatalog). A series of intensive econometric analysis is conducted on a regression on used car prices against variables of engine capacity, odometer value, newness, car age etc, with new techniques applied such as bootstrapping, robust estimation and running a Boruta Algorithm to indicate the best variables for estimation.
