Machine Learning: Marketing

Machine learning model building for marketing dataset on restaurant coupon acceptance rates using Python.

Using data from the UCI machine learning repository (Available at: https://archive.ics.uci.edu/ml/datasets/in-vehicle+coupon+recommendation), I build and evaluate 4 different classification models to predict In Vehicle Coupon Acceptance Rates for restaurants from marketing data surveyed through Amazon Mechanical Turk. The 4 models include: Logistic Regression, Lasso Regression, Ridge Regression and a Kernel Support Vector Machine (SVM). The hyperparameters for the latter three machine learning techniques are cross validated and chosen based on ideal testing performance, before all four models are evaluated based on their respective confusion matrices, predictive accuracy and ROC AUC values. Results show that an SVM with a polynomial kernel is the most ideal model to make predictive classifications on whether an agent accepts a restaurant coupon or not.

This project was done using Python through Jupyter Notebook, giving me practical experiences in building machine learning classification models that can be applied in future projects.
