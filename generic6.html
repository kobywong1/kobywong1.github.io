Kaggle Project: House Prices

Coursework Project. Regression analysis of data found on Kaggle involving house prices in Iowa using R.

Using data from kaggle (Available at: https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data), I regress house prices against 10 chosen explanatory variables. Interestingly enough, results show that variables for sale year are irrelevant, while variables for living area, housing quality/condition, garage capacity and the year the house was built are tested to be significant.

This project was conducted using R, giving me experience in coding using R Markdown, as well as experiences in coding advanced econometric techniques such as Mallow's CP, n-fold Cross Validation tests, transformation to linearity by interpreting Box-Cox graphs and residual analysis, just to name a few. The extensive and rigorous econometric tests on model design have strengthened my understanding of statistical analysis, and would allow me to apply these techniques in future projects.
